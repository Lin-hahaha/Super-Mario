# Super Mario (超级玛丽) 🍄

这是一个使用 Java (AWT/Swing) 编写的“超级玛丽”重制版小游戏。项目还原了经典的平台跳跃玩法，包含多关卡设计、原生像素级图片以及背景音效。

## ✨ 游戏特色
* **经典还原**：原汁原味的跳跃系统、顶金币、踩怪物逻辑与移动碰撞检测。
* **多关卡设计**：
  * **第一关**：经典的长水管、平坦地面与旗杆城堡的组合。
  * **第二关**：场景难度提升，加入了短高间隔水管、地面坠落深坑，以及更危险的敌人排布。
* **敌人系统**：包含会左右巡逻的“小蘑菇”和在水管中上下伸缩的“食人花”。
* **音效支持**：内置了原生游戏的 `.wav` 音乐体验（需配备 `jl-1.0.1.jar`）。

## 🚀 环境要求与运行
* **环境配置**：需要 Java 8 或更高版本的 JDK。
* **依赖库**：使用了 `javazoom.jl.player.Player` 播放音频，需要将项目根目录下的 `jl-1.0.1.jar` 添加到你的项目库 (Libraries / Classpath) 中。
* **游戏入口**：
  运行 `src/com/sm/GameWin.java` 中的 `main()` 方法即可启动游戏。

## 🎮 操作说明
* `A` / `←`（左箭头）：向左跑
* `D` / `→`（右箭头）：向右跑
* `W` / `↑`（上箭头）：跳跃

## 🛠️ 项目结构
* `src/com/sm/GameWin.java`: 游戏核心主窗口，控制游戏主线程循环与场景绘制。
* `src/com/sm/BackGround.java`: 背景控制器，负责当前关卡的场景切换、障碍物和敌人的初始化。
* `src/com/sm/Mario.java`: 主角马里奥的实体类，封装了运动、跳跃和碰撞检测逻辑。
* `src/com/sm/Enemy.java`: 敌人控制类。
* `src/com/sm/Obstacle.java`: 障碍物（各种水管、砖块、旗杆）的物理属性控制。
* `src/com/sm/Music.java`: 使用 `javazoom` 实现本地音频的后台播放。
* `src/com/sm/GameUtil.java`: 静态资源文件工具，在游戏启动时负责统一加载图像等资源。
